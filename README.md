Micro module to help eliminate one of the [bad typewriter habits][habits].


## Install

```sh
npm install --save typographic-trademark
```


## Usage

```js
var trademark = require('typographic-orphan');

trademark(`2013 MegaCorp (TM)`); // 2013 MegaCorp™
```
